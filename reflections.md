# Self-Reflections Log

Track self-reflections from completed work.
Use this file for concise local reflection summaries and promotion notes.
Store detailed reflection episodes in Pinecone DEEP memory.

## Format

```markdown
## [Date] - [Task Type]

What I did: brief summary
Outcome: success | partial | failed
Reflection: what was learned
DEEP Event: <pinecone event id>
Promotion: pending | promoted | archived
```

## Example Entry

```markdown
## 2026-04-04 - API Refactor

What I did: refactored interceptor and retry strategy
Outcome: success
Reflection: single-flight locking prevented duplicate token refresh
DEEP Event: default#reflection#20260404T172000400#7c2d9aef
Promotion: pending (needs 3 confirmations)
```

## Rules

- Keep entries short and factual.
- Link to DEEP event ids for full detail.
- Promote only repeated, validated lessons into Markdown rules.
